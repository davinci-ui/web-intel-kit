# GenSpark Tools Reference

## URL Structure
- Home: `https://www.genspark.ai/`
- No direct URLs for individual tools — all accessed via prompt bar or tool icons on homepage

## Prompt Patterns

### AI Slides
- "Create AI Slides about [topic]"
- "Create a presentation comparing [X] vs [Y]"
- Generates 10-12 slides with professional design
- Output: interactive slide deck with Preview/Code/Thinking tabs

### AI Docs
- "Create a document about [topic]"
- "Write a report on [subject]"
- Output: formatted document

### AI Sheets
- "Create a spreadsheet tracking [data]"
- "Build a comparison table of [items]"
- Output: interactive spreadsheet

### AI Designer
- "Design a logo for [brand]"
- "Create a banner for [event]"
- Output: graphic design

### AI Chat (Super Agent)
- Direct questions or complex multi-step requests
- Super Agent researches, analyzes, then responds
- Can chain multiple tools together

### AI Image
- "Generate an image of [description]"
- Output: AI-generated image

## Account & Credits
- GenSpark uses a credit system
- Free tier includes limited credits
- Pro plan available with "Switch to Annual and save 20%" offer
- Credits visible on homepage (look for "credits" text)
- Survey completion earns 5,000 free credits

## Authentication
- Login via Google OAuth or email
- Session persists in browser profile
- Cookie-based auth (gslogin=1 when logged in)
